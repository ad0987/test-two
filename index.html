4. Locally weighted linear regression and bias-variance tradeoff. (25 points)¶

(a) (5 points) Show that the ridge regression which introduces a squared l2 norm penalty on the¶

parameter in the maximum likelihood estimate of β can be written as follows βˆ(λ) = arg min 􏰀(Xβ − y)T W (Xβ − y) + λ∥β∥2􏰁

$$\hat \beta(\lambda) = \arg\min_{\beta}\left\{\frac 1 n \sum_{i=1}^n (y_i -x_i^T \beta)^2 + \lambda \|\beta\|_2^2\right\}$$

(b) (5 points) Find the close-form solution for β(λ) and its distribution conditioning on {xi}.¶

For inimizing the beta function, taking the derivative of $\hat\beta(\lambda)$ with respect to $\beta$ and setting it equal to zero. Then we solve for $\hat\beta$:

$$\frac{\partial{\hat \beta(\lambda)}}{\partial{\beta}} = \arg\min_{\beta}\left\{-\frac{2}{n}Xy + \frac{2}{m}XX^T\beta + \frac{2\lambda}{m}\beta\right\} = \arg\min_{\beta}\left\{-2X^T(y - \beta^TX) + 2\lambda\beta\right\} = 0$$

Step2 will be solving the equation for $\hat\beta$:

$$\hat\beta(\lambda) = (X^TX + \lambda I)^{-1} X^T y$$

with $y$ is the random variable.

$$\mathbb E[\hat\beta] = (X^TX + \lambda I)^{-1} X^T X \beta^*$$

Here $\beta^*$ denotes the real parameters.

$$Var[\hat\beta] = \sigma^2 (X^TX + \lambda I)^{-1} X^T X [(X^TX + \lambda I)^{-1}]^T$$

The random variable $y$ is Gaussian, since the error term $\epsilon$ is Gaussian. Then consequently $\hat\beta$ is Gaussian. So:

$$\hat\beta \sim \mathcal{N}(\mathbb E[\hat\beta],Var[\hat\beta]) $$

(c) (5 points) Derive the bias as a function of λ and some fixed test point x.¶

For deriving the bias as a function of λ we take the Expectation fro the above equation as below: $$Bias = \mathbb E[x^T {\hat \beta}(\lambda)] - x^T {\beta^*} = \mathbb E[x^T {(X^TX + \lambda I)^{-1} X^T y}] - x^T {\beta^*}$$ Based on probability theory, for the random variable y and for any $\alpha$, the following is true:$$\mathbb E[\alpha y] = \alpha\mathbb E[y] = \alpha X \beta^*$$Thus the equation above becomes:

$$x^T \mathbb E[{(X^TX + \lambda I)^{-1} X^T y}] - x^T {\beta^*} = x^T {(X^TX + \lambda I)^{-1} X^T X {\beta^*}} - x^T {\beta^*}$$

So based on the above questions the Bias will be equal to : $$x^T {(X^TX + \lambda I)^{-1} X^T X {\beta^*}} - x^T {\beta^*}$$

(d) (5 points) Derive the variance term as a function of λ.¶

In a previous step we calculated the $Var[\hat\beta]$. Substituting the variance in the equation we get :

$$Var[x^T\hat\beta] = x^T Var[\hat\beta] x= x^T \sigma^2 (X^TX + \lambda I)^{-1} X^T X [(X^TX + \lambda I)^{-1}]^T x$$

So variance will be $$x^T \sigma^2 (X^TX + \lambda I)^{-1} X^T X [(X^TX + \lambda I)^{-1}]^T x$$
